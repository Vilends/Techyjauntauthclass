# Techyjaunt Auth + Tasks API (CRUD + JWT)

RESTful API built with Node.js, Express.js, MongoDB, JWT, and bcrypt.
Includes user authentication (register/login) and protected CRUD operations for Tasks.

## Features
- Register user (bcrypt hashed password)
- Login user (JWT token)
- JWT middleware protects task routes
- Tasks CRUD (create, read, update, delete)

## Tech Stack
Node.js, Express.js, MongoDB, Mongoose, JWT, bcryptjs, morgan, dotenv
